# Contributing to Resonance Field Theory — Numerical Demonstration

Notes on contributing to the numerical toolkit.

---

## Reporting Bugs

- First check the [Issues](https://github.com/DominicReneSchu/public/issues)
- Create a new issue with Python version, OS and error message

## Code Contributions

1. Fork & Clone
2. Create a branch: `git checkout -b improvement`
3. Write code, add tests
4. `python tests/test_numerical_demonstration.py` must pass
5. Pull request with description

## Style

- PEP 8
- Type hints and docstrings
- Tests for new functions
- Document axiom references in docstrings

## Scientific Extensions

- New ε models (e.g. non-linear coupling efficiency)
- Multi-dimensional parameter spaces
- Experimental data integration
- Discuss in an issue beforehand
