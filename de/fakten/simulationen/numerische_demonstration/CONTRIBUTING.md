# Beitrag zur Resonanzfeldtheorie — Numerische Demonstration

Hinweise zur Mitarbeit am numerischen Toolkit.

---

## Fehler melden

- Prüfe zuerst die [Issues](https://github.com/DominicReneSchu/public/issues)
- Neuen Issue mit Python-Version, OS und Fehlermeldung erstellen

## Code-Beiträge

1. Fork & Clone
2. Branch erstellen: `git checkout -b verbesserung`
3. Code schreiben, Tests ergänzen
4. `python tests/test_numerische_demonstration.py` muss durchlaufen
5. Pull Request mit Beschreibung

## Style

- PEP 8
- Type Hints und Docstrings
- Tests für neue Funktionen
- Axiom-Bezug in Docstrings dokumentieren

## Wissenschaftliche Erweiterungen

- Neue ε-Modelle (z.B. nichtlineare Kopplungseffizienz)
- Mehrdimensionale Parameterräume
- Experimentelle Datenanbindung
- Vorab in Issue diskutieren
