## 1. Klassische Mechanik — Schwingungen als Grundlage

### Anerkannte Fakten

**Hookesches Gesetz und harmonischer Oszillator:** Die Rückstellkraft eines
elastischen Systems ist proportional zur Auslenkung:

```
F = −kx   →   mx'' + kx = 0   →   ω = √(k/m)
```

Jedes elastische System schwingt mit einer charakteristischen Eigenfrequenz ω,
die durch seine physikalischen Parameter bestimmt wird.

**Gekoppelte Pendel:** Wenn zwei Pendel gleicher Eigenfrequenz mechanisch
verbunden werden, ist der Energieaustausch zwischen ihnen maximal. Bei
unterschiedlichen Eigenfrequenzen nimmt der Energieaustausch ab. Die Resonanz
ist die Bedingung für maximalen Energietransfer.

**Erzwungene Schwingung:** Wenn ein Oszillator durch eine äußere periodische
Kraft angeregt wird, ist die Amplitude maximal, wenn die Anregungsfrequenz
der Eigenfrequenz entspricht:

```
Amplitude  ∝  1 / √[(ω² − ω₀²)² + (γω)²]
→  Maximum bei ω = ω₀  (Resonanzbedingung)
```

**Stehende Wellen:** Auf einer Saite oder in einer Röhre entstehen stehende
Wellen nur bei rationalen Frequenzverhältnissen. Die erlaubten Frequenzen sind
ganzzahlige Vielfache der Grundfrequenz:

```
fₙ = n · f₁   (n ∈ ℕ)   →   rationale Verhältnisse fₙ/fₘ = n/m
```

**Doppelpendel:** Chaotisch im Grenzfall, aber der Energietransfer zwischen den
Gliedern folgt der Phasendifferenz θ₂ − θ₁. Die Kopplungseffizienz ist maximal
bei Δθ = 0.

### RFT-Verbindung

```

(A3)  Resonanzbedingung: |f₁/f₂ − m/n| < δ  =  Bedingung für max. Energieaustausch.
(A4)  Resonanzkurve der erzwungenen Schwingung = Kopplungseffizienz ε(Δφ):
      Je kleiner die Phasendifferenz, desto größer die übertragene Amplitude.

Die mechanische Resonanzkurve *ist* die Kopplungseffizienz. Die RFT benennt
das Prinzip, das die Mechanik bereits kennt.

## 2. Thermodynamik — Entropie als Resonanzverlust

### Anerkannte Fakten

**Zweiter Hauptsatz:** In einem abgeschlossenen System nimmt die Entropie S
nie ab. Geordnete Zustände gehen spontan in ungeordnete über.

**Boltzmann-Entropie:** Die Entropie misst die Anzahl der Mikrozustände W,
die einem Makrozustand entsprechen:

```
S = k_B · ln(W)
```

Je mehr Mikrozustände möglich sind, desto größer die Entropie — desto
ungeordneter das System.

**Wärmeübertragung:** Wärme fließt spontan nur von heiß nach kalt (ΔT als
Antrieb). Temperatur ist die mittlere kinetische Energie der Teilchen:

```
T = (2/3) · ⟨E_kin⟩ / k_B   →   E_kin ∝ T   →   mittlere Schwingungsenergie
```

**Carnot-Wirkungsgrad:** Der maximale Wirkungsgrad einer Wärmekraftmaschine
zwischen zwei Temperaturen T_kalt und T_warm ist:

```
η_Carnot = 1 − T_kalt / T_warm
```

Da E = k_B · T und E = h · f gilt: T ∝ f. Damit ist der Carnot-Wirkungsgrad
ein Frequenzverhältnis:

```
η_Carnot = 1 − f_kalt / f_warm = (f_warm − f_kalt) / f_warm
```

Der Wirkungsgrad ist maximal, wenn die Frequenzdifferenz maximal ist.

**Phasenübergänge:** Kristallisation, Bose-Einstein-Kondensat: Entropie sinkt
*lokal*. Ordnung entsteht spontan, wenn Teilchen eine gemeinsame Phase finden.

### RFT-Verbindung

Thermodynamik beschreibt, was passiert, wenn Resonanz verloren geht oder sich
bildet:

```

      Die Oszillatoren schwingen zunehmend außer Phase — ε(Δφ) → 0.

(A3)  Phasenübergang = spontane Resonanzbildung:
      Oszillatoren finden plötzlich eine gemeinsame Phase — ε → 1.

(A4)  Kopplungseffizienz ε(Δφ) = cos²(Δφ/2):
      analog zum Carnot-Wirkungsgrad — maximale Effizienz bei minimaler
      Phasendifferenz (f_kalt → f_warm entspricht Δφ → 0).
```

## 3. Elektrodynamik — Felder als Schwingungen

### Anerkannte Fakten

**Maxwell-Gleichungen:** Die vier Maxwell-Gleichungen beschreiben elektrische
Felder **E** und magnetische Felder **B** als gekoppelte, sich gegenseitig
erzeugende Größen. Aus ihnen folgt die Wellengleichung:

```
∂²E/∂t² = c² · ∇²E
```

Elektromagnetische Wellen sind Schwingungen des elektromagnetischen Feldes.

**Elektromagnetische Wellen:** E und B schwingen senkrecht zueinander und in
Phase, senkrecht zur Ausbreitungsrichtung. Die Wellenlänge λ und Frequenz f
bestimmen alle Eigenschaften:

```
c = λ · f
```

**Impedanzanpassung:** Maximale Energieübertragung zwischen Quelle und Last
erfolgt, wenn die Impedanzen übereinstimmen:

```
Z_Quelle = Z_Last   →   maximale Leistungsübertragung, keine Reflexion
```

**RLC-Schwingkreis:** Resonanzfrequenz eines elektrischen Schwingkreises:

```
f₀ = 1 / (2π√(LC))   →   maximale Amplitude bei f = f₀
```

**Antennen:** Senden und Empfangen ist nur effizient bei Resonanz. Eine
λ/4-Antenne ist auf die Wellenlänge λ der Sendefrequenz abgestimmt:
Resonanzbedingung als Designprinzip.

(A6)  Radio empfängt nur den eingestellten Sender:

## 4. Quantenmechanik — Quantisierung durch Resonanz

### Anerkannte Fakten

**Planck-Relation:** Energie ist quantisiert und proportional zur Frequenz:

```
E = h · f   (Planck 1900, h = 6,626 × 10⁻³⁴ J·s)
```

**De-Broglie-Wellenlänge:** Jedem Teilchen mit Impuls p ist eine Wellenlänge
zugeordnet:

```
λ = h / p   →   jedes Teilchen hat eine charakteristische Frequenz
```

**Schrödinger-Gleichung:** Die Zeitentwicklung einer Wellenfunktion ψ(x,t):

```
iℏ ∂ψ/∂t = Ĥψ
```

**Atomorbitale:** In einem Atom sind nur stehende Wellen erlaubt — nur
diejenigen Wellenfunktionen, die ganzzahlige Vielfache der Grundwellenlänge

→   E_n = −13,6 eV / n²   (Wasserstoff)
→   rationale Frequenzverhältnisse f_n/f_m = n²/m²

## 5. Relativitätstheorie — Geometrie der Resonanz

**Spezielle Relativitätstheorie:** Kein absolutes Bezugssystem. Die
Lichtgeschwindigkeit c ist in allen Inertialsystemen gleich. Gleichzeitigkeit
ist relativ.

**Allgemeine Relativitätstheorie:** Masse krümmt die Raumzeit. Die Metrik
beschreibt die Geometrie der gekrümmten Raumzeit:

```
ds² = g_μν dx^μ dx^ν
```

**Gravitationswellen (LIGO 2015):** Zwei verschmelzende Schwarze Löcher

steigende Amplitude, bis zur Verschmelzung.

beschreibt das expandierende Universum mit dem Skalenfaktor a(t):

```
ds² = −c²dt² + a(t)² [dr²/(1−kr²) + r²dΩ²]
